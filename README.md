# cfo-industrial-mindset
📊 CFO Industrial Mindset

Este projeto aplica conceitos de IA na Indústria e Métodos Computacionais para Finanças (ICMC-USP) para modelar uma empresa como um sistema produtivo.

A proposta é simples, mas poderosa:

👉 tratar uma empresa como se fosse uma fábrica
👉 e suas linhas de negócio como um portfólio

💡 O que é feito neste projeto
Análise exploratória de dados (EDA)
Medição de variabilidade (risco)
Cálculo de retorno por linha de negócio
Construção de um “Sharpe corporativo”
Simulação de cenários financeiros
Otimização do mix de receitas
